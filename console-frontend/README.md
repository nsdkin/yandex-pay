# Yandex Pay Console

## Требования
- NodeJS v14
- NPM v6

## Подготовка
Создать `<repo-root>/.env` и положить в него `OAUTH_TOKEN=<toke-data>`. 

## Быстрый старт
- `npm install`
- `npm run dev`
- Заходим на [http://localhost:3000](http://localhost:3000)
